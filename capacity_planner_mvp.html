3. Calculations and Definitions

Metric Formula Used Description
Available Memory per Cluster (AMC) $$ \text{Nodes per Cluster} \times \text{Available Memory per Node} $$ Total available RAM across all nodes in the cluster.
Total Storage available per cluster (TSAPC) $$ \text{NPC} \times \text{DPN} \times \text{Device Size} \times (1 - \text{Cluster Storage Overhead Pct}) $$ The effective total usable storage capacity after system overhead.
Total Data Stored (TDS) $$ \text{Master Object Count} \times \text{Replication Factor} \times \text{Average Record Size (converted to GB)} $$ Total size of user data across the cluster, including replication.
Actual Storage Utilization (ASU) $$ \text{Total Data Stored} / \text{Total Storage Available per Cluster} $$ Ratio of data stored to available capacity. Key for provisioning.
Primary Index Shmem Used (PISU) $$ \frac{\text{MOC} \times \text{RF} \times 64 \text{ bytes per entry}}{1024^3} $$ Estimated memory for the Primary Index. (64 bytes per PI entry assumed).
Secondary Index Shmem Used (SISU) $$ \frac{(\text{MOC} \times \text{RF} \times \text{Avg SI entries per object} \times 21)}{0.75 \times 1024^3} + (\text{Number of SI} \times 16 \text{ MiB to GB}) $$ Estimated memory for Secondary Indexes, including entry overhead and per-index overhead.
Total Memory Used (TMU) $$ \text{Overhead Memory Required} + \text{PISU} + \text{SISU} $$ Total memory usage without accounting for tombstone overhead.
Total Memory w/ Tombstones (TMT) $$ (\text{PISU} + \text{SISU}) \times (1 + \text{Tombstone Pct}) + \text{Overhead Memory Required} $$ Worst-case memory usage, including overhead from durable deletes.
Total Memory Utilization w/ Tombstones (TMUT) $$ \text{Total Memory w/ Tombstones} / \text{Available Memory per Cluster} $$ Key performance indicator for memory pressure.
