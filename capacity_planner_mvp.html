4. Calculations and Definitions

Metric Formula Used Description
Available Memory per Cluster (AMC) Nodes per Cluster × Available Memory per Node Total available RAM across all nodes in the cluster.
Total Storage available per cluster (TSAPC) Nodes per Cluster × Devices per Node × Device Size × (1 - Cluster Storage Overhead Pct) The effective total usable storage capacity after system overhead.
Total Data Stored (TDS) Master Object Count × Replication Factor × Average Record Size (B) / 1024^3 Total size of user data across the cluster, including replication.
Actual Storage Utilization (ASU) Total Data Stored (TDS) / Total Storage Available (TSAPC) Ratio of data stored to available capacity. Key for provisioning.
Primary Index Shmem Used (PISU) (Master Object Count × Replication Factor × 64 bytes) / 1024^3 Estimated memory for the Primary Index. (64 bytes per PI entry assumed).
Secondary Index Shmem Used (SISU) (MOC × RF × Avg SI entries × 21 bytes / 0.75) / 1024^3 + (Number of SI × 16 MiB / 1024) Estimated memory for Secondary Indexes, including entry overhead and per-index overhead.
Total Memory Used (TMU) Overhead Memory Required (OMRC) + PISU + SISU Total memory usage without accounting for tombstone overhead.
Total Memory w/ Tombstones (TMT) (PISU + SISU) × (1 + Tombstone Pct) + Overhead Memory Required (OMRC) Worst-case memory usage, including overhead from durable deletes.
Total Memory Utilization w/ Tombstones (TMUT) Total Memory w/ Tombstones (TMT) / Available Memory per Cluster (AMC) Key performance indicator for memory pressure.
